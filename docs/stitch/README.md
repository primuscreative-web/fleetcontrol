# Google Stitch Reference

Place the exported Google Stitch files in this directory.

The Stitch export is the official source for UX/UI decisions. Implementation work must map its visual identity into the FleetControl design system without redesigning screens.

Recommended mapping targets:

- `apps/web/src/app/globals.css`
- `apps/web/tailwind.config.ts`
- `apps/web/src/lib/design-tokens.ts`
- `apps/web/src/components/*`
